# rust_text_editor
rust_text_editor for ТСПО in ПИШ РТУ МИРЭА

### Запуск проекта:
1) Склонить репозиторий:
```bash
git clone https://github.com/CyberGeo335/rust_text_editor.git
```
2) перейти в корневую директорию:
```bash
cd rust_text_editor/ && cargo build
```
3) Если ошибки, выполняем следующее:
```
cargo install cargo-edit

cargo add eframe --features persistence
cargo add egui
cargo add rfd

cargo build
```
